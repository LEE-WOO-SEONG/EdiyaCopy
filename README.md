# IdiyaCopy



## 4.3 회의결과

1. 네이밍 방식
   - 케밥케이스를 베이스로 클래스 네이밍을하되 음료의 특정 네이밍을 할떄에는 카멜케이스로한다.
2. 네이밍 규칙
   - a = link
   - ul = list
   - li = item
3. 기능
   - 헤더고정
   - 이미지 확대
   - 메뉴아이콘 클릭시 메뉴등장
   - 메뉴닫기 아이콘 확대
   - 메뉴 버튼 hover bg color
   - 음료설명 팝업
   - 음료설명 팝업 닫기 버튼 백그라운드
   - 음료설명 팝업 닫기 버튼 아웃라인 애니메이션
   - 음료 마크업 css? html?
4. 모듈
   - 숨김컨텐츠
   - 리셋
5. 추가기능
   - 메뉴 들여쓰기 & 크기 조정
   - 음료설명 bg filter
   - to top 버튼
   - 메뉴설명 닫기 버튼이 너무 작음
   - 설명 툴팁
   - 네비게이션 바 첫번쨰 메뉴 수정
